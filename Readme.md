Тестовый проект по автоматизации тестирования мобильного приложения Alchemy.

Платформа Android 11 API30. Java 22

В проекте используются тестовые библиотеки JUnit5, Selenide-Appium.

Для создания детализированных, интерактивных и визуально понятных отчетов о результатах автоматизированного тестирования подключен Allure.

Для успешного запуска автотестов приложение Alchemy должно быть установлено на эмулятор.
